The goal of "Octocat's data lab" is to provide the visual analysis of the GitHub data to millions of users throughout the world using which they can plan their future.


 Main Functionality of “Octocat's data lab” is,

1. Language Correlation:
To analyze the data from GitHub and show user the correlation between various programming languages.
2. Geek Planet:
To display the visual representation of the data being committed to GitHub from various parts of the world.
3. Coding Mood:
What is the mood of the developer while working with different programming languages?
4. Hottest Technologies:
What hottest technologies are currently in the market?
5. Language Trends:
Total number of commits made for a particular programming language on a given day.
